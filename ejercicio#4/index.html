titulo segundo nivel

HTML son las siglas en inglés de HyperText Markup Lenguage
que significa Lenguaje de Marcado de HiperTexto.
Se llama así al lenguaje de programación empleado en la elaboración de páginas Web

titulo #2 segundo nivel

JavaScript es un lenguaje de programación interpretado,
dialecto del estándar ECMAScript. Se define como orientado a objetos,
basado en prototipos, imperativo, débilmente tipado y dinámico.
